# Ginansya

Ginansya is an Android finance application developed using Java in Android Studio. The application was created to help users visualize how their savings can grow over time through compound interest and monthly contributions.

The main goal of the application is to provide a simple and beginner-friendly way for users to monitor their savings progress and understand potential financial growth without needing internet connectivity or online banking integration.

## Current Features

- Dashboard screen showing overall savings overview
- Savings plans with progress tracking
- Compound interest calculator with live computation
- Real-time projected savings computation
- Monthly contribution input
- Projected vs actual savings chart
- Financial insights and analytics
- Interactive charts and visualizations
- Offline functionality with locally stored temporary data
- Modern Material Design 3 user interface


## Current Progress

The current version of the application already includes the main user interface, navigation system, financial calculator, charts, and sample savings plans. The application is currently focused on improving the logic and interaction features such as adding plans, tracking contributions, and monitoring savings progress.

## Notes

This project is intended for educational purposes only and does not connect to real banking systems or online services.
